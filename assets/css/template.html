Close Panel

.close-panel

Closes the panel. Put inside the .panel but outside heading, body, or footer.

<span class="close-panel"><i class = "fa fa-times fa-fw"></i></span>

Needs this jQuery code to work

$('.close-panel').click(function() {
	$(this).parent().css('display','none');
});
