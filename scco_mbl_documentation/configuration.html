How To Set Up

Configuration are filles that is essential to deploying the web application

Table of Contents

  • Config.php
  • Database.php

Config.php

Config.php is the main configuration file for the web application. See CodeIgniter’s documentation for further knowledge

Config.php can be found at ssco_mbl/application/config/config.php

Database.php

Database.php is used to initialize the bridge between the server and the client. It contains credentials of the server as well as the database the web app is using

Database.php can be found at ssc_mbl/application/config/database.php
