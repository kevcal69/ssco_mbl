Controllers

Controllers are the heart of your application, as they determine how HTTP requests should be handled.A Controller is simply a class file that is named in a way that can be associated with a URI.

Home Controller

File name: home.php

Location:ssco_mbl/application/controllers/home.php

The page outlet. This controllers the landing page and login page

Search Controller

File name: search.php

Location:ssco_mbl/application/controllers/search.php

This controller manages module searches from the user. The search is tag based for modules.

Session Controller

File name: session.php

Location:ssco_mbl/application/controllers/session.php

The session controller, manages the log in log out of accounts.

Admin Controller

File name: admin.php

Location:ssco_mbl/application/controllers/admin/admin.php

The admin controller redirect the user upon log in to landing page

Module Controller

File name: module.php

Location:ssco_mbl/application/controllers/admin/module.php

This controller controls all admin functions concerning the modules.

Question Controller

File name: question.php

Location:ssco_mbl/application/controllers/admin/question.php

This controller controls all admin functions concerning about module test questions and schedule test question

Test Controller

File name: test.php

Location:ssco_mbl/application/controllers/admin/test.php

This controller manages the admin functions concerning test results statistics.

Trainee Controller

File name: trainee.php

Location:ssco_mbl/application/controllers/admin/trainee.php

This controller manages admin's operation of the trainees. Admin can view trainees, enrol or unenrol to modules and view trainee's test results.

User Controller

File name: module.php

Location:ssco_mbl/application/controllers/admin/user.php

This controller manages users as general.

Module Controller

File name: module.php

Location:ssco_mbl/application/controllers/admin/trainee.php

This controller manages all actions the trainee can do about modules. It also manages all views of modules requested by the trainee
